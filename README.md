# Advanced Portfolio Website

A modern and responsive personal portfolio website built using HTML5, CSS3, and JavaScript. This project demonstrates advanced CSS concepts including CSS Grid, Flexbox, CSS Variables, Animations, Responsive Design, and BEM methodology.

## 🚀 Features

- Responsive Design
- CSS Grid Layout
- Flexbox Components
- CSS Variables
- Smooth Animations & Transitions
- BEM Naming Convention
- Contact Form
- Mobile Friendly
- Clean UI

## 🛠 Technologies Used

- HTML5
- CSS3
- JavaScript
